

# 🤖 Teachable Machine Project (Machine Learning + Deep Learning)

## 📌 Overview

This project is built using **Google Teachable Machine**, combining **Machine Learning (ML)** and **Deep Learning (DL)** to create an intelligent model capable of recognizing and classifying inputs such as images, sounds, or poses.

The goal of this project is to demonstrate how AI can be easily trained without complex coding and applied to real-world classification problems.

---

## 🚀 Features

* 🧠 Model built using Teachable Machine
* 📸 Image/Sound/Pose classification
* 🤖 Deep Learning-based prediction
* ⚡ Real-time detection capability
* 📊 Simple and efficient training process
* 🌐 Exportable model for web/app integration

---

## 🧠 About Teachable Machine

**Teachable Machine** is a web-based tool by Google that allows users to train machine learning models without writing extensive code.

It supports:

* Image classification
* Audio recognition
* Pose detection

---

## ⚙️ Technologies Used

* 🐍 Python
* 🤖 TensorFlow / Keras
* 🌐 Teachable Machine
* 📷 OpenCV (for real-time detection, if used)
* 📦 NumPy

---

## 🔬 Workflow

1. **Data Collection**

   * Collected images/sounds/poses for each class

2. **Model Training**

   * Trained model using Teachable Machine platform

3. **Model Export**

   * Exported model in TensorFlow/Keras format (`.h5`)

4. **Integration**

   * Integrated model into Python using Keras

5. **Prediction**

   * Real-time or static input classification
